клас img
display: block; - дозволяє картинці займати весь оступний простір по ширині (картинка стає блоком)

клас page wrapper
max-width: 860px; - обмеження максимальної ширини для контейнера сторінки 
width: 100%; - блок займає всю ширину екрану 
padding: 30px 10px 20px; - встановлення внутрішніх відступів навколо контенту 

блок header
background-repeat: no-repeat; - при умові, ящко блок більший за картинку, зображення не буде дублюватись 
background-size: contain - масштабує картинку так, щоб вона поміщалась в блок не обрізаючись 
background-position: center; - потрібне для центрування фонового зображення 

блок main
padding: 20px 30px; - створює простір усередині основної частини сайту.
font-size: 1.2rem; - встановлення розміру шрифту відносно базового розміру браузера

клас list
padding-left: 60px; - створення відступу зліва для списку 

клас services-list-item
width: calc(50% - 15px); - розрахунок ширини елемента так, щоб у рядку помістилось рівно 2 блоки (15 пікселів для того, щоб врахувати половину відступу)

клас services-list-item__title
padding: 15px 0; - створення відступів зверху і знизу по 15 пікселів

клас footer p:last-child a - вибирає всі посилання, які знаходяться всередині останнього параграфа
